> Исследовательский анализ данных о пассажирах Титаника

## Датасет
- Источник: [Kaggle: Titanic Dataset](https://www.kaggle.com/datasets/yasserh/titanic-dataset)
- Записей: 891 пассажир
- Целевая переменная: `Survived` (0/1)

## Технологии
- Python 3
- pandas, numpy, matplotlib, seaborn
- Google Colab

## Ключевые выводы
1. Женщины выживали в 3.5× чаще мужчин (74% vs 19%)
2. 1-й класс: 63% выживаемости, 3-й класс: 24%
3. Дети имели приоритет: 58% выживаемости
4. Стоимость билета коррелирует с выживаемостью (r=0.26)
5. Пассажиры с семьёй: 43% vs 30% у одиноких
